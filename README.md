Some files are saved as .xlsm file, also known as an Excel Macro-Enabled Workbook. To use those files properly, you need to unblock the macros by modifying the properties of the file as follows:

1. Open Windows File Explorer and navigate to the folder where you saved the file.

2. Right-click the file and choose Properties from the context menu.

3. At the bottom of the General tab, select the Unblock checkbox and then click OK.
